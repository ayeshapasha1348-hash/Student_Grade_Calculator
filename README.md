# Student Grade Calculator

## 1. Project Overview
The Student Grade Calculator is a beginner-level Python project.  
It takes a student's name and marks as input, calculates the grade (A, B, C, D, F), and displays an encouraging message based on the grade.  
This project helps beginners practice **if-elif-else statements, loops, functions, and input validation**.

---

## 2. Project Objectives
- Learn basic Python programming concepts  
- Understand and use **if-elif-else statements**  
- Use **while loops** for input validation  
- Create **functions** for reusable code  
- Display formatted and friendly output  
- Practice creating a **GitHub repository** with documentation
